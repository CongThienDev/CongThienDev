## Profile

Fullstack web developer building production React/Next.js frontends and
Node.js/PostgreSQL backends, with hands-on experience in **payment integrations,
microservices on Kubernetes, and technical SEO / web performance**.

I ship real products: commerce frontends with live Stripe/Klarna/PayPal checkout,
event-driven backend services, and sites with Lighthouse SEO consistently at 90–100.

## Core Strengths

- **Frontend:** React, Next.js (App Router), TypeScript, Tailwind CSS, GSAP / Framer Motion, responsive UI
- **Backend:** Node.js, Express, REST APIs, PostgreSQL, Redis, BullMQ workers
- **Payments:** PayPal SDK (sandbox → live), Stripe via dedicated payment service, Klarna checkout
- **Microservices & DevOps:** service-to-service calls in Kubernetes, Docker / Docker Compose, Nginx, GitHub Actions CI/CD, staging → production promotion
- **Technical SEO:** dynamic sitemaps, robots, JSON-LD schema, hreflang, Open Graph, Core Web Vitals
- **Quality:** Vitest unit tests, Playwright E2E, ESLint / Prettier, Husky pre-commit hooks
